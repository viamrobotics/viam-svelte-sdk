# viam-svelte-sdk

Reactive Svelte 5 hooks and components for building apps with Viam-powered machines. Wraps [`@viamrobotics/sdk`](https://github.com/viamrobotics/viam-typescript-sdk) with TanStack Query, runes, and provider components.

```bash
pnpm i --save @viamrobotics/svelte-sdk @viamrobotics/sdk @tanstack/svelte-query
```

## Documentation

Full documentation, including hook references, components, and a live playground, lives at:

**https://viamrobotics.github.io/viam-svelte-sdk/**
